Project Overview

The ANAV system was developed for autonomous exploration in GPS‑denied environments emulating Martian conditions. It combines stereo vision, LiDAR, and IMU sensor fusion with real‑time decision making on an NVIDIA Jetson Orin Nano to perform SLAM, safe spot detection, and emergency landing without external positioning.

Key Algorithms

RTAB‑Map, ICP, Dijkstra, PointNet++, ResNet-based classification

Key Features

Safe‑spot detection, lost‑link fallback, low battery emergency landing, ROS2 + PX4 integration

Performance

30s stable hover, landing accuracy ≈ 30 cm, reliable lost‑link recovery

Testing

PX4 SITL, Gazebo (Mars‑like terrain), real‑world qualification & elimination rounds

Summary

Built by Team Xenith (Kumaraguru College of Technology). Demonstrated robust autonomy in competition and advanced to the IRoC-U elimination round. The system is modular, production‑ready for further research, and extensible with additional sensors such as long‑range LiDAR.

ANAV — Autonomous Navigation Aerial Vehicle. Selected Top 37 in IRoC-U 2024. GPS‑denied Martian‑like autonomy using Jetson Orin Nano + Pixhawk, MYNT EYE 3D, RPLiDAR S1. Algorithms: RTAB‑Map, ICP, Dijkstra, PointNet++, ResNet. Features: safe‑spot detection, lost‑link/low‑battery recovery.

Tech Stack Used

NVIDIA Jetson Orin Nano Pixhawk 2.4.8 (PX4) MYNT EYE 3D / RealSense D455 SLAMTEC RPLiDAR S1 ROS2 Python OpenCV PointNet++, ResNet Gazebo / PX4 SITL
